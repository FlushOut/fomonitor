Documentação de erros

Para acessar a documentação de métodos, clique aqui.

  1. 101 Você está tentando acessar um método inexistente.
  2. 102 Vocé precisa passa usuário e senha para os métodos que exigem login. Exemplo: {URL}/login/coduser/1234/password/swordfish .
  3. 103 Você não tem permissão para acessar este método.
  4. 104 Para usar o método licencekey, você precisa enviar na URL os parâmetros usuário, senha e chave. Exemplo: Exemplo: {URL}/applicence/coduser/1234/password/swordfish/licencekey/9876 .
  5. 105 Para usar o método getapp, você precisa enviar na URL os parâmetros usuário, senha e código da aplicação. Exemplo: Exemplo: {URL}/getapp/coduser/1234/password/swordfish/codapp/555 .
  6. 106 Para usar o método location, você precisa enviar na URL os parâmetros usuário, senha, latitude, longitude, velocidade (em metros por segundo) e data (yyyymmddhhmmss). Exemplo: Exemplo: {URL}/location/coduser/1234/password/swordfish/lat/99.999/lon/99.999/speed/99.999/datetime/20130820215512 .
  7. 107 O parâmetro lat (latitude) deve conter um valor do tipo float válido. Exemplo: 99.999
  8. 108 O parâmetro lon (longitude) deve conter um valor do tipo float válido. Exemplo: 99.999
  9. 109 O parâmetro speed deve conter um valor do tipo float válido, representando metros por segundo. Exemplo: 99.999
  10. 110 O parâmetro datetime deve estar no formato yyyymmddhhmmss. Exemplo: 20130820215512 para 20/08/2013 21h55m12s
